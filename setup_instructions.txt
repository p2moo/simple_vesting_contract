//How to Create a New Smart Contract

1. Open terminal

2. Select Ubuntu

3. Navigate to folder you want to create the new contract

4. Type in the terminal
```
anchor init _ProjectName_
```

5. Do stuff

6. Type in terminal
```
git init
git remote add origin _UrlOfGit_
git add .
git commit -m "comment"
git push  --set-upstream origin master
```
